# Elevator-Dispatch-Algorithm
基于机器视觉与多源信息融合的电梯群控调度算法（本科毕设）

---

## 项目简介
本项目为本科毕业设计的完整代码与仿真结果，实现了基于混合颗粒度GA-PSO的电梯调度算法，并与传统固定分区调度算法进行对比验证，旨在提升高层建筑电梯群控系统的运行效率与节能效果。

## 核心内容说明
1. **算法设计**
- 对照组：传统固定分区电梯调度算法（按楼层均分服务区域）
- 实验组：本文提出的基于混合颗粒度GA-PSO的电梯调度算法，引入多源信息融合提升决策精度

2. **仿真与对比实验**
- 场景：12层建筑、2台电梯的早高峰客流模拟
- figure1：本文优化算法收敛曲线，迭代3次左右即快速收敛，适应度值稳定
- figure2：调度分区对比，优化算法打破了固定分区限制，可动态分配高客流楼层
- figure3：性能指标对比，优化方案在乘客等待时间、系统能耗上均显著优于传统方案
- figure4：普通GA-PSO算法收敛曲线，存在明显震荡，稳定性弱于本文优化算法

## 运行环境
- 编程语言：MATLAB
- 运行要求：MATLAB R2018b及以上版本，无需额外工具箱

---

## 项目亮点
- 提出混合颗粒度GA-PSO算法，兼顾全局优化效率与局部调度精度
- 相比传统固定分区调度，显著降低乘客等待时间与系统能耗
- 对比普通GA-PSO算法，收敛更快、稳定性更强，无明显震荡
